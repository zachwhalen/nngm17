Introduction

This is a book about (), , and the many, many things that have something to do with these houses. Inside, you can see their pictures and imagine their adventures.

I created this book in November 2017 as my entry in "NaNoGenMo" (National Novel Generation Month), which is an event where people write computer programs that generate novels. To make this book, I wrote a program in the Python language that used information from ConceptNet.io and The Noun Project to figure out how certain words relate to each other and what they should look like.

All together, this book contains about words!

You can find more detail and the complete source code at github.com/zachwhalen/nngm17. To learn more about me or get in touch, check out my website at www.zachwhalen.net.
